## Como executar o projeto

### Instalar as dependências
```bash
npm install
```

### Executar o projeto
```bash
npm run start:dev
```

### Executar os testes unitários
```bash
npm run test
```

### Executar os testes em modo watch

```bash	
npm run test:watch
```
